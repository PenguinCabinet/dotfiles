# dotfiles of PenguinCabinet
dotfiles of PenguinCabinet.

# How to Install

1. Install (Ubuntu Server 22.04 LTS)[https://ubuntu.com/download/server](minimum)
2. Run commands
```
sudo apt install git -y
git clone https://github.com/PenguinCabinet/dotfiles
cd dotfiles 
sudo bash install.sh
```
